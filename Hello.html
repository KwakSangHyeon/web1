HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

The United States of Massachusetts Institute of Technology (MIT) and the European Atomic Nucleus Joint Research Institute (CERN), which developed Web servers, are involved in standardization, such as Microsoft and Netscape Communications (communications). The group is progressing the standardization of the hypertext Generating language HTML, the Hypertext Transfer Protocol (HTTP), the communication protocol between the Web browser/server, the page description language of the WWW (PDL: page Description language). The W3C WebVR community Group promotes web-based virtual reality standardization.
